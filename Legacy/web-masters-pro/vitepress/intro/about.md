# О шаблоне


## Технологический стек

Основные технологии, используемые в проекте, включают [Docker](https://www.docker.com/) для контейнеризации приложений, [Laravel 12](https://laravel.com/) для разработки бэкенда, [Nuxt 3](https://nuxt.com/) для создания современного фронтенда, в качестве базы данных выступает [MySQL 8](https://www.mysql.com/), веб-сервер работет на [Nginx 1.21](https://nginx.org/ru/) и документация работающая на [VitePress](https://vitepress.dev/ru/).
