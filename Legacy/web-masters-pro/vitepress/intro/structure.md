# Файловая структура проекта

Файловая структура проекта организована таким образом, чтобы четко разделить компоненты приложения, соответствующие различным контейнерам Docker. В корневом каталоге находятся пять основных папок:

## Laravel `(/laravel)`

Папка содержит все файлы и директории, относящиеся к бэкенду проекта, написанному на [Laravel](https://laravel.com/). Основные элементы этой папки:

- `app/`: Содержит классы контроллеров, моделей и сервисов.
- `config/`: Хранит конфигурационные файлы Laravel.
- `database/`: Включает миграцию баз данных, семантику и фабрику данных.
- `lang/`: Языковые файлы
- `public/`: Корневая директория веб-сервера, содержащая статические файлы (CSS, JS, изображения).
- `routes/`: Определение маршрутов для RESTful API и веб-интерфейсов.
- `storage/`: Хранилище данных (кэш, логи, сессии)
- `tests/`: Тестовые сценарии PHPUnit
- `composer.json`: Управляет зависимостями PHP-проекта.
- `Dockerfile`: Конфигурационный файл Laravel-образа 
- `php.ini`: Настройки php применяемые в контейнере Laravel 
- `artisan`: Исполняемый файл консольной утилиты Artisan 
 

## Nuxt `(/nuxt)`

Папка включает в себя все файлы и директории, относящиеся к фронтенду проекта, созданного с использованием [Nuxt.js](https://nuxt.com/). Основные элементы этой папки:

- `assets/`: Статические ресурсы, такие как стили, шрифты и изображения.
- `components/`: Компоненты Vue.js, которые используются в приложении.
- `layouts/`: Шаблоны макетов страниц.
- `pages/`: Маршруты и страницы приложения.
- `plugins/`: Плагины Vue.js.
- `server/`: Серверные функции и API.
- `stores/`: Хранилища [Pinia](https://pinia.vuejs.org/).
- `nuxt.config.ts`: Основной конфигурационный файл Nuxt.js
- `package.json`: Управляет зависимостями JavaScript-проекта.
- `app.vue`: Главный компонент приложения. Это точка входа для рендеринга приложения.
- `Dockerfile`: Конфигурационный файл Nuxt-образа 
- `entrypoint.sh`: Точка входа в контейнер Nuxt

## VitePress `(/vitepress)`

::: warning 
Контейнер доступен только в dev-собрке проекта.
:::

Папка включает в себя все файлы и директории, относящиеся к документации проекта, созданного с использованием [VitePress](https://vitepress.dev/ru/). Основные элементы этой папки:

- `.vitepress/`: Основная директория для конфигурации VitePress. Она содержит файлы, которые управляют поведением и внешним видом вашего сайта.
- `.vitepress/config.mjs`: Главный конфигурационный файл. Здесь настраиваются тема, навигация, боковая панель, плагины и другие параметры.
- `.vitepress/theme/`: Директория темы
- `public/`: Директория для статических файлов, таких как изображения, шрифты или favicon.
- `index.md`: Главная страница
- `package.json`: Управляет зависимостями JavaScript-проекта.
- `Dockerfile`: Конфигурационный файл VitePress-образа 
- `entrypoint.sh`: Точка входа в контейнер VitePress

## MySQL `(/mysql)`

Папка предназначена для хранения настроек базы данных. Основные элементы этой папки:

- `conf.d/my.cnf`: Конфигурационный файл MySQL.

## Nginx `(/nginx)`

Папка содержит конфигурационные файлы Nginx, которые управляют проксированием запросов между различными сервисами. Основные элементы этой папки:

- `nginx.conf`: Основной конфигурационный файл Nginx.
